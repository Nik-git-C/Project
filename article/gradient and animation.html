Работа с градиентами и анимациями

Градиент Анимации

Градиент

linear-gradient()

linear-gradient(to bottom, red,blue);

Структура: linear-gradient((at/to) Направление,color1,color2)

linear-gradient(90deg, red, red 20px,blue 20px,blue 50px,orange 50px);

Направление можно завать разными спосбами, если после цвета можно прописать где он находится в рахных единицах измерения

radial-gradient()

background: radial-gradient(at right,red,blue);

Структура:radial-gradient((at/to) Направление,color1,color2); свойства такие же как и линейного градиента

background: radial-gradient(circle at 65% 15%,#fff,red);

repeating-radial-gradient()

background: repeating-radial-gradient(circle, red,red 20px, transparent 20px,transparent 40px);

Анимации

transition:

transition: - уневерсальное собираетльное свойство которое включает в себя другие, для динамических изменений

transition: 2s background steps(10);

2s - время на выполнение, background - свойство которое изменяется постепенно, steps(количество) - прыжки или кадры

animation:

animation: - уневерсальное собираетльное свойство которое включает в себя другие, для изменений прописанные в @keyframes

animation: color 6s infinite;
@keyframes color {
    0% {
        border-color: red;
        background: blue;
    }
    20% {
        border-color: blue;
        background: red;
        border-radius: 100%;
    }
    40% {
        border-color:orange;
        background: green;
        border-radius: 0;
    }
    60% {
        border-color:red;
        background: grey;
        border-radius: 100%;
    }
    100% {
        border-color:green;
        background: black;
        border-radius: 0;
    }
}

color - имя анимации, 6s - время анимации, infiniteм - анимация повторяется, @keyframes прописываются кадры которой следует анимация
